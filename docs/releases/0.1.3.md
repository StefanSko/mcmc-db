# mcmc-ref v0.1.3

Release date: TBD

## Summary

Consolidation release that makes the provenance pipeline the canonical path and removes remaining hardcoded jaxstan source paths.

## Highlights

- Added explicit generator metadata to provenance manifests:
  - `generator.name`
  - `generator.version`
  - `generator.source_commit`
- Marked legacy import/conversion commands as deprecated:
  - `mcmc-ref-build-references`
  - `mcmc-ref-import-informed`
  - `scripts/sync_stan_models.py`
- Removed hardcoded `/tmp/jaxstanv3/...` defaults from runtime tooling.
- Updated integration and release docs so `provenance-scaffold -> provenance-generate -> provenance-publish` is the primary maintainer workflow.
- Aligned versions for this release line:
  - `mcmc-ref==0.1.3`
  - `mcmc-ref-data==0.1.3`
