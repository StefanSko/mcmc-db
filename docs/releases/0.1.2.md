# mcmc-ref v0.1.2

Release date: TBD

## Summary

Patch release focused on reparametrization pair reliability and data quality.

## Highlights

- Added bundled reference artifacts for `bangladesh_contraceptive-noncentered`:
  - `draws/bangladesh_contraceptive-noncentered.draws.parquet`
  - `meta/bangladesh_contraceptive-noncentered.meta.json`
- Artifacts are generated from real CmdStan sampling (`source: cmdstan-2.37.0`).
- Added regression coverage ensuring bundled pairs have usable reference draws/stats.
- Added regression coverage enforcing bundled pair reference metadata source starts with `cmdstan-`.
- Updated contributor policy in `AGENTS.md`:
  - bundled references must come from real CmdStan runs
  - synthetic/mock bundled reference artifacts are not allowed
