# mcmc-ref 0.1.1

Release date: 2026-02-15

## Highlights

- Added 5 missing benchmark models to the packaged `mcmc-ref-data` corpus:
  - `wells_data-wells_dist`
  - `GLM_Binomial_data-GLM_Binomial_model`
  - `GLM_Poisson_Data-GLM_Poisson_model`
  - `radon_mn-radon_hierarchical_intercept_noncentered`
  - `irt_2pl-irt_2pl`
- Included full asset sets for those models in `mcmc-ref-data`:
  - `draws/*.draws.parquet`
  - `meta/*.meta.json`
  - `stan_data/*.data.json`
  - `stan_models/*.stan`
- Added packaging regression coverage to ensure these assets remain in the
  distributable corpus.

## Versioning

- `mcmc-ref`: `0.1.1`
- `mcmc-ref-data`: `0.1.1`

uv add "mcmc-ref[data]==0.1.1"
